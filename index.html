Eclipse Decentralized Claims Protocol

This document defines a base protocol for communicating participant identities and claims in a Tractus-X dataspace. This specification assumes familiarity with the [Tractus-X Dataspace Topology Specification](tx.dataspace.topology.md).

This document is currently under revision as it is being transferred from the Eclipse Tractus-X organization to the governance and maintenance of the Eclipse Dataspace Working Group.
